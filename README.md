## Running Locally

- Make sure you have Node >= 18 installed.
- Rust and Cargo installed
- Install dependencies: `pnpm i` (or any other package manager like `yarn` or `npm`)
- Start worker: `pnpm dev`
- Make HTTP call: `curl http://localhost:8787`
